# PrepPal — AI Study Tools

AI-powered study companion with flashcards, quizzes, summaries, homework help, poster generator, and focus timer.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your GitHub repo
3. Vercel auto-detects Vite — just click Deploy
4. Done! Your site is live.

## Local Development

```bash
npm install
npm run dev
```
